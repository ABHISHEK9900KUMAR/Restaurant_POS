# Dinefy — Feature Roadmap

## High Impact (Customer-facing)
- [x] Special instructions per item — "Less spicy / no onion" note per cart item
- [x] Favourite items — Heart icon, localStorage, "My Favourites" quick filter
- [x] Combo / add-on suggestions — "Pair with Dal Tadka?" when adding a main course
- [x] Item availability countdown — Admin sets qty limit; "Only 3 left" badge
- [x] Estimated wait time — Cook time per item; tracker shows "Ready in ~N min"

## Critical (Legal / Business Viability)
- [x] Cash payment tracking — "Mark Paid (Cash)" on orders, 💵 cash chip in header, cash vs UPI in report
- [x] GST billing — GSTIN + SAC 996331 on receipt, CGST 2.5% + SGST 2.5% rows, sequential invoice numbers (FS-YYYY-NNNN), GST Settings panel in admin
- [x] GST-formatted daily report — CGST/SGST breakdown, cash vs UPI split in both PDF and HTML report

## Admin / Operations
- [ ] Kitchen Display System (KDS) — `/kitchen` live order board
- [ ] Daily sales summary — `/admin/report` with revenue, top items
- [ ] Call-waiter button — Customer tap → admin real-time notification
- [ ] Scheduled offers — Start/end time, auto-activates

## Retention / Loyalty
- [ ] Phone-based loyalty points (server-side, per phone number)
- [ ] SMS/WhatsApp order confirmation via Twilio
